phy++ documentation
Signature 
bool empty(string s)

Description
This function will return 
true
if the provided string does not contain any character at all (inluding spaces), and 
false
otherwise.

Example
vec1s str = {"", "abc", "   "};
vec1b b = empty(str);
b; // {true, false, false}
// Not to be confused with the vec::empty() function
str.empty(); // false
str = {""};
str.empty(); // false
str = {};
str.empty(); // true
