vec<D,int_t>  indgen(...)

vec<D,uint_t> uindgen(...)

vec<D,float>  findgen(...)

vec<D,double> dindgen(...)

Description
These functions will create a new vector, whose dimensions are specified in argument like in the dimension constructor SEC:core:vec:constructor or 
vec::resize()
SEC:core:vec:member_fun. After this vector is created, the function will fill it with values that start at 
0
and increment by steps of 
1
until the end of the vector.

Example
vec1i v = indgen(5);    // {0,1,2,3,4}
vec2u w = uindgen(3,2); // {{0,1}, {2,3}, {4,5}}
