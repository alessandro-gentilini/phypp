string keep_start(string s, uint_t n = 1)

string keep_end(string s, uint_t n = 1)

Description
These functions will return the first (
keep_start
) or last (
keep_end
) 
n
characters of the string 
s
and discard the rest. If 
n
is larger than the size of 
s
, the whole string is returned untouched.

Example
vec1s v = {"p1_m2.txt", "p3_c4.fits", "p1_t8.fits"};
vec1s s = keep_start(v, 2);
s; // {"p1", "p3", "p1"}
s = keep_end(v, 4);
s; // {".txt", "fits", "fits"}
