Signature 
bool file::is_older(string f1, string f2)

Description
This function returns 
true
if the file or directory 
f1
is older than the file or directory 
f2
. The age of a file corresponds to the time spent since the files were last modified. If one of the two files does not exists, the function returns 
false
.

Example
bool b = file::is_older("~/.phypprc", "/usr/bin/cp");
b; // probably false
