vec shuffle(auto& seed, vec v)

void inplace_shuffle(auto& seed, vec& v)

Description
These functions will randomly modify the order of all the values inside the vector 
v
. The function 
shuffle()
will re-order a copy of the provided vector, while 
inplace_shuffle()
will modify the provided vector directly and is therefore faster. Note that there is a non-zero probability that the order will remain the same.

Example
// We create the seed (see make_seed())
auto seed = make_seed(42);

vec1u v = indgen(5);
v; // {0,1,2,3,4}

inplace_shuffle(v);
v; // could be {3,1,2,4,0}
