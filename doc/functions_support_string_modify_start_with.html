bool start_with(string s, string p)

bool end_with(string s, string p)

Description
These functions will return 
true
if the beginning (
start_with
) or the end (
end_with
) of the string 
s
is exactly the same as the string 
p
.

Example
vec1s v = {"p1_m2.txt", "p3_c4.fits", "p1_t8.fits"};
vec1b b = start_with(v, "p1");
b; // {true, false, true}

b = end_with(v, ".fits");
b; // {false, true, true}
