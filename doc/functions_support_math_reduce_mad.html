T mad(vec<D,T> v)

vec<D-1,T> partial_mad(uint_t d, vec<D,T> v)

Description
The function 
mad()
computes the median absolute deviation of the values in 
v
, i.e., 
median(fabs(v - median(v)))
. Note that, for a set of values with a distribution close to a Gaussian, the median absolute deviation is just the standard deviation divided by . Like the median compared to the mean, it is however less sensitive to strong outliers.

Calling 
mad()
on an empty vector will trigger an error.

The function 
partial_mad()
will apply 
mad()
on the 
d
th dimension of the vector (zero being the first dimension) and reduce its number of dimensions by one.

Example
vec1d v = {-1,1,0.5,2,1.5};
mad(v); // 0.5
