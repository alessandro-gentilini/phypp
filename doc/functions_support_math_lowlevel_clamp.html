Signature 
T clamp(T t, mi, ma)

Description
This function will compare its first argument against 
mi
and keep the largest value. Then, it will compare this latter value against 
ma
and return the smallest of both. In other words, the goal of this function is to make sure that a given value 
t
is bound to a specified interval 
[mi, ma]
. It is a more readable equivalent to 
min(max(t, mi), ma)
.

Example
vec1d x = {0.0, -0.05, 1.0, 0.5, 0.2, 1.06, 0.95, dnan};
// Make sure that the input values are between 0 and 1
clamp(x, 0.0, 1.0); // {0.0,0.0,1.0,0.5,0.2,1.0,0.95,nan}
