uint_t where_first(vec<D,bool>)

uint_t where_last(vec<D,bool>)

Description
These functions will scan the 
bool
vector provided in argument, and return the flat index (\ref{SEC:core:vec:indexing}) of the first (or last) element which is 
true
, and 
npos
if all are 
false
.

Example
vec1i v = {4,8,6,7,5,2,3,9,0};
// We want to select the first element which is greater than 3
uint_t id;
id = where_first(v > 3); // 0
v[id];                   // 4
id = where_last(v > 3);  // 7
v[id];                   // 9
