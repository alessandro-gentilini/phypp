double fraction_of(vec<D,bool> v)

vec<D-1,double> partial_fraction_of(uint_t d, vec<D,bool> v)

Description
The function 
fraction_of()
returns the number of values in 
v
that are 
true
divided by the total number of values.

Calling 
fraction_of()
on an empty vector returns not-a-number.

The function 
partial_fraction_of()
will apply 
fraction_of()
on the 
d
th dimension of the vector (zero being the first dimension) and reduce its number of dimensions by one.

Example
vec1d v = {false,false,true,true,false};
fraction_of(v); // 0.4

vec2d w = {{false,true,true,false}, {false,false,true,false}};
fraction_of(0, w); // {0,0.5,1,0}
