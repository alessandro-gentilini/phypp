phy++ documentation
Signature 
string remove_extension(string)

Description
This function scans the provided string to look for a file extension (e.g., ".fits"). The extension is whatever ends the string after the last dot (and including this dot). If one is found, the function will return a new string with the extension removed. Else, the input string is returned untouched.

Example
vec1s v = {"p1_m2.txt", "p3_c4.fits", "p1_t8.dat.fits", "readme"};
vec1s s = remove_extension(v);
s; // {"p1_m2", "p3_c4", "p1_t8.dat", "readme"}
