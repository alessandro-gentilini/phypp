string remove_extension(string)

string get_extension(string)

pair<string> split_extension(string)

Description
These function scan the provided string to look for a file extension (e.g., ".fits"). The extension is whatever ends the string after the last dot (and including this dot). If one is found, the function 
get_extension()
will return this extension (including the dot) while the function 
remove_extension()
will return the input string with the extension removed. Else, 
get_extension()
returns an empty string, while 
remove_extension()
returns the input string untouched. The function 
split_extension()
does both at the same time and returns a pair containing the trimmed file name as 
first
and the extension as 
second
.

Example
vec1s v = {"p1_m2.txt", "p3_c4.fits", "p1_t8.dat.fits", "readme"};
vec1s s = remove_extension(v);
s; // {"p1_m2", "p3_c4", "p1_t8.dat", "readme"}

s = get_extension(v);
s; // {".txt", ".fits", ".fits", ""}
