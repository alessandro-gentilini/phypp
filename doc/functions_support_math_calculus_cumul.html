Signature 
vec<1,T> cumul(vec<1,T> x, y)

Description
This function computes the cumulative integral of the tabulated function . The return value is a vector whose value 
v[i]
corresponds to the integral of from 
x[0]
to 
x[i]
. The first value is therefore always zero, while the last value is equal to 
integrate(x,y)
. The function assumes that the vector 
x
is sorted and only contains unique values (no check is performed).

Example
vec1d x = rgen(-3.0, 3.0, 10); // build a grid of 10 points
vec1d y = exp(-x*x);           // sample a function

vec1d r = cumul(x, y);
// 'r' now holds something that is close to the formal integral of
// the Gaussian function, i.e., the error function, evaluated at each
// point 'x'.
