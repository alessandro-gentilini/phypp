vec1u mult_ids(vec v, uint_t i)

vec1u mult_ids(array<uint_t> dims, uint_t i)

Description
This is only useful for multidimensional vectors. This function converts a “flat” index 
i
(\ref{SEC:core:vec:indexing}) into an array of multidimensional indices, following the dimensions of the provided vector 
v
(or, in the second version, following the dimensions 
dims
). The 
flat_id
function does the inverse job.

Example
vec2i v(2,3);
mult_ids(v,0); // {0,0}
mult_ids(v,1); // {0,1}
mult_ids(v,2); // {0,3}
mult_ids(v,3); // {1,0}
v[3] == v(1,0); // true
