Signatures 
vec<1,T> remove(vec<1,T> v, vec1u id)

void inplace_remove(vec<1,T>& v, vec1u id)

Description
These functions will remove the elements in 
v
that have their indices in 
id
. The only difference between the first and the second version is that the former will first make a copy of the provided vector, remove elements inside this copy, and return it. The second version modifies directly the provided vector, and is therefore faster.

Example
// First version
vec1i v = {4,5,2,8,1};
vec1i w = remove(v, {1,3}); // {4,2,1}

// Second version
inplace_remove(v, {1,3});
v; // {4,2,1}
