Signatures 
double weighted_mean(vec v, w)

double weighted_median(vec v, w)

Description
The functions 
weighted_mean()
(and 
weighted_median()
) computes the weighted average (median) of the values in 
v
, each weighted by a corresponding value in 
w
.

Calling 
weighted_mean()
on an empty vector returns not-a-number, and calling 
weighted_median()
on an empty vector will trigger an error.

Example
vec1d v = {-1,1,0.5,2,1.5};
vec1d w = {0.0,1.0,0.5,2.0,1.0}
weighted_mean(v); // 1.7222
weighted_median(v); // 0.5
