Signature 
bool is_any_of(T1 v1, vec<D2,T2> v2)

Description
This function looks if there is any value inside 
v2
that is equal to 
v1
. If so, it returns 
true
, else it returns 
false
.

Example
vec1i v = {7,4,2,1,6};
vec1i d = {5,6,7};
vec1b b = is_any_of(v, d); // {true, false, false, false, true}
