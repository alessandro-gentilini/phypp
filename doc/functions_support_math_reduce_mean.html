Signatures 
double mean(vec v)

pair<double> optimal_mean(vec v, e)

vec<D-1,double> partial_mean(uint_t d, vec<D,T> v)

Description
The function 
mean()
computes the average of the values in 
v
, i.e., the sum divided by the number of elements. The function 
optimal_mean()
will compute the average of the values in 
v
(
pair.first
) optimally weigthed by their respecive uncertainties provided in 
e
, and also return the uncertainty on the average (
pair.second
). This weighting, using the inverse square of the uncertainties, provides the optimal for the average value.

Calling these functions on an empty vector(s) returns not-a-number.

The function 
partial_mean()
will apply 
mean()
on the 
d
th dimension of the vector (zero being the first dimension) and reduce its number of dimensions by one.

Example
vec1d v = {-1,1,0.5,2,1.5};
mean(v); // 0.8

vec2d w = {{0,1,2,3,4}, {0,0,1,3,6}};
partial_mean(0, w); // {0,0.5,1.5,3,5}

vec1d e = {1,3,100,2,1};
std::pair<double,double> om = optimal_mean(v, e);
om.first;  // average:     0.470589
om.second; // uncertainty: 0.650778
