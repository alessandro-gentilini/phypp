Signature 
uint_t length(string s)

Description
This function will return the length of the provided string, i.e., the number of character it contains (including spaces). If the string is empty, the function will return zero.

Example
vec1s str = {"", "abc", " a b"};
vec1u n = length(str);
n; // {0, 3, 4}
