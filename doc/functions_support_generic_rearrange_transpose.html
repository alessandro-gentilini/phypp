Signature 
vec<2,T> transpose(vec<2,T>)

Description
This function will transpose the provided vector so that its dimensions are swapped, i.e., 
v(i,j)
becomes 
v(j,i)
.

Example
vec2i v = {{1,2}, {3,4}, {5,6}};
vec2i w = transpose(v); // {{1,3,5}, {2,4,6}}
// now w(i,j) == v(j,i)
