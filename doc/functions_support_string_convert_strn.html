string strn(T v)

string strn_sci(T v)

Description
These functions will convert the value 
v
into a string. This value can be of any type, as long as it is convertible to string. In particular, 
v
can be a vector, in which case the output string will contain all the values of the vector, separated by commas, and enclosed inside curly braces 
"{...}"
.

Example
strn(2);            // "2"
strn(true);         // "1"
strn("foo");        // "foo"
strn(vec1i{2,5,9}); // "{2, 5, 9}"

The second version is dedicated to floating point numbers, and will output them in scientific format.

Example
strn_sci(2.0);    // "2.000000e+00"
strn_sci(2e10);   // "2.000000e+10"
strn_sci(-5e-2);  // "-5.000000e-02"

// Integer numbers are not affected
strn_sci(1000);   // "1000"
strn_sci(1000.0); // "1.000000e+03"
