phy++ documentation
Signature 
vec1u equal_range(T x, vec v)

Description
Similarly to 
lower_bound
, 
upper_bound
and 
bounds
, this function uses a binary search algorithm to locate all the values in the input vector 
v
that are equal to 
x
. It then returns the flat indices of all these values in a vector. It is important to note that the binary search assumes that the elements in the input vector are sorted by increasing value. This algorithm also assumes that the input vector does not contain any NaN value (\ref{SEC:support:math}).

If no such value is found, an empty vector is returned.

Example
vec1i v = {2,2,5,9,9,9,12,50};
equal_range(9, v); // {3,4,5}

// It's a faster version of
where(v == 9);
