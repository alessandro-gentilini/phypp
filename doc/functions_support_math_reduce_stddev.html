double stddev(vec<D,T> v)

vec<D-1,double> partial_stddev(uint_t d, vec<D,T> v)

Description
The function 
stddev()
computes the standard deviation of the values in 
v
, i.e., 
sqrt(mean(sqr(v - mean(v))))
.

Calling 
stddev()
on an empty vector returns not-a-number.

The function 
partial_stddev()
will apply 
stddev()
on the 
d
th dimension of the vector (zero being the first dimension) and reduce its number of dimensions by one.

Example
vec1d v = {-1,1,0.5,2,1.5};
stddev(v); // 1.02956
