phy++ documentation
Signature 
uint_t find(string s, p)

Description
This function returns the position in the string 
s
of the first occurrence of the sub-string 
p
. If no such occurrence is found, the function returns 
npos
.

Example
vec1s v = {"Apple", "please", "complementary", "huh?"};
vec1u p = find(v, "ple");
p; // {2, 0, 3, npos}
