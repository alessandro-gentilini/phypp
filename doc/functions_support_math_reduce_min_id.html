uint_t min_id(vec v)

uint_t max_id(vec v)

pair<uint_t> minmax_ids(vec v)

Description
The functions 
min_id()
and 
max_id()
return respectively the index of the minimum and maximum values in 
v
. The function 
minmax_ids()
does both at the same time. Calling them on an empty vector will trigger an error.

Example
vec1d v = {-1,1,0.5,2,1.5};
min_id(v); // 0
max_id(v); // 3
