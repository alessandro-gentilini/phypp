Signature 
int_t sign(T)

Description
This function will return the sign of its argument. The most convenient way to achieve this is by returning an integer whose value is 
+1
for positive (or null) numbers, and 
-1
for strictly negative numbers. This allows convenient usage of this function inside more complex mathematical expressions without having to write conditional statements.

The result of calling this function on “not a number” values is undefined.

Example
// A stupid example.
vec1d x = {-1.0, 0.0, -1000.5, 20.0, 5.0};
// By computing the square of 'x', we loose its sign.
vec1d x2 = x*x;
// Say we want to get back to 'x'
// If we write: x2 = sqrt(x2)
// ... the value is right, but not the sign.
// Instead we can write:
x2 = sign(x)*sqrt(x2);
x2; // {-1.0,0.0,-1000.5,20.0,5.0}
