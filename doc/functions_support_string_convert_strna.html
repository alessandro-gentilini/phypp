Signatures 
vec_<D,string> strna(vec<D,T>)

vec_<D,string> strna_sci(vec<D,T> v)

Description
These functions are the vectorized version of 
strn
. The reason why the name of the function is different is because it is already possible to call 
strn
with vector arguments: the whole vector will be converted into a single string. These versions however convert each element separately to form a vector of strings.

Example
strna(vec1i{1,5,6,9}); // {"1", "5", "6", "9"}
strn(vec1i{1,5,6,9});  // "{1, 5, 6, 9}"
