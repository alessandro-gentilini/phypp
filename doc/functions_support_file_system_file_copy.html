Signature 
bool file::copy(string from, to)

Description
This function creates a copy of the file 
from
at the location given in 
to
and returns 
true
. If the new file could not be created (e.g., because of permission issues), or if the file 
from
could not be found or read, the function returns 
false
. If the file 
to
already exists, it will be overwritten.

Example
bool b = file::copy("/home/joe/.phypprc", "/home/bob/.phypprc");
b; // maybe true or false, depending on your permissions
