Signature 
void file::find_skip(string f)

Description
This function scans the file whose path is given in the string 
f
and returns the number of lines at the beginning of the file that either: a) are empty, b) only contain spaces or tabs, or c) start with a 
'#'
(which is the de facto standard for comments in ASCII tables). The return value is typically fed to 
file::read_table()
.

Example
// Improving the first example of file::read_table().

// We first compute the number of lines to skip
uint_t nskip = file::find_skip("some/ascii_table.dat");

// Declare the vectors and their types
vec1u id;
vec1d ra, dec;

// Then read the data
file::read_table("some/ascii_table.dat", nskip,
    id, ra, dec // read three columns
);
